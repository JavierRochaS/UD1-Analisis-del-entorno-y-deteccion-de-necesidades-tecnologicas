## Introducción
Este repositorio contiene la documentación inicial para el diseño de una infraestructura informática segura y centralizada. El proyecto parte de un análisis del entorno tecnológico en **Sevilla (Andalucía)** y se aplica a una empresa del sector logístico en **Coria del Río**.
